# Quartzo

### Segundo cérebro em Markdown — local-first, rápido, offline.

Um PKM (gerenciador de conhecimento) desktop: escreva em Markdown, conecte ideias com `[[wikilinks]]`, veja o **grafo** da sua base, desenhe no **canvas** e volte a qualquer nota pelos **backlinks**. Suas notas, seus arquivos, sua máquina — **sem nuvem obrigatória, sem conta**.

## 🚧 Em desenvolvimento

Quartzo **ainda não foi lançado** — está em desenvolvimento ativo. Esta página é uma prévia pública do que está sendo construído. O código-fonte é privado; aqui mostramos só a cara e a direção do produto. Quando sair, o download vai aparecer aqui.

## O que é

Local-first de verdade: tudo vira **arquivo Markdown na sua pasta** — nada de formato fechado, nada de servidor. Abre offline, salva sozinho, e você leva suas notas pra onde quiser.

## O que já roda

- ✍️ Editor Markdown fluido (modo edição / dividido / leitura)
- 🕸️ **Grafo** da base + **Canvas** infinito + **backlinks**
- 🔗 `[[wikilinks]]`, transclusão `![[nota]]`, tags e busca
- 📊 Views dinâmicas (tabela / kanban / tarefas), Mermaid, KaTeX
- 🎨 Temas (a identidade é um cristal de quartzo — cyan sobre navy)
- 🕘 Versões locais (Git nativo), sem depender de nuvem
- 🖥️ Windows e **macOS** (build universal Intel + Apple Silicon)

## Como vai funcionar (open-core)

- **Grátis pra sempre:** o núcleo completo — notas, grafo, canvas, backlinks, busca, temas claro/escuro.
- **Pro (planejado):** sincronia com Google Drive, exportar (DOCX / PDF / HTML) e temas extras. Compra única, sem mensalidade.
